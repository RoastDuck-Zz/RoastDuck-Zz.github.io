项目属性： 全栈独立开发
核心技术栈： ESP32-S3 双板架构 | OpenCV 人脸检测追踪 | 小智 AI 语音交互 | 4G Cat.1 物联网 | 总线舵机云台

本项目源于对嵌入式 AI 终端的探索，基于双 ESP32-S3 主控架构，深度集成了小智 AI 大模型与 OpenCV 实时人脸检测追踪算法。与传统的 Wi-Fi 桌面小电视不同，这台桌宠板载了 ML307R 4G Cat.1 模组，通电即联网，使其成为一个真正意义上全天候在线的独立 AI 终端。视觉方面，ESP32-S3-CAM 节点负责捕捉人脸特征并引导 2-DOF 云台进行目标追踪，实现了「你在哪，桌宠就看向哪」的拟真互动。

# 💻 硬件架构与选型拆解

这个桌宠拥有完整的感知、思考与表达能力，硬件架构主要分为以下几个核心域：

# 1. 核心计算与视觉双枢纽

  • 主控大脑：ESP32-S3-DevKitC-1
    负责整个系统的资源调度。它需要同时处理 I2S 麦克风录音流、解析小智 AI 下发的音频并驱动扬声器，同时还要通过 AT 指令与 4G 模块高频通信，S3 的双核算力在此处被压榨到了极致。
  • 视觉感知：ESP32-S3-CAM
    独立的视觉协处理器。负责捕捉桌面环境与人脸特征，解析出人物坐标后，引导下方的两自由度云台进行追踪，实现“你在哪，桌宠就看向哪”的拟真互动感。该主板上装载了0V3660摄像头模块,OV3660具备300万像素,支持硬件JPEG编码,可以输出最高15fps的2K分辨率图像。此外0V3660还具备OV5660等不具备的功能---夜视。使用0V3660的桌宠即使在光线较暗的环境下也能清晰的捕捉到人脸特征,可以更好地满足监控的需求.

# 2. 交互与表达外设

  • 显示输出：LCD 屏幕
    镶嵌在桌宠头部的“脸”。主要用于渲染动态的 UI 交互界面与拟人化面部表情（如眨眼、开心、思考等），让冰冷的机器具备情绪反馈。
  • 听觉收集：INMP441 全向麦克风
    采用 I2S 数字接口的 MEMS 麦克风，相比于传统的模拟麦克风，它的底噪极低、保真度高，能非常清晰地捕捉桌面周围的语音指令，极大提升了小智 AI 语音识别（ASR）的准确率。

# 3. 仿生脖颈（2-DOF 云台）

为了让桌宠具备灵动的头部姿态，颈部采用了幻尔总线舵机组搭建的两自由度（2-DOF）云台结构：

  • Pan（水平环顾）： 底座采用 HTS-25L 大扭矩总线舵机，负责承载整个头部的重量，提供平滑且宽广的水平旋转视角。
  • Tilt（垂直点头）： 关节处采用 HX-08L 轻量化总线舵机，响应速度极快，负责实现桌宠的“点头”、“抬头”等高频动作交互。

# 4. 全天候网络中枢：ML307R 4G Cat.1

  • 放弃了不稳定的局域网 Wi-Fi，接入中国移动物联网 ML307R 模组。这使得桌宠可以摆脱路由器的束缚，无论放在办公桌、床头还是带出门，通电即联网，随时随地与云端的大模型保持长连接。

# 📌 后续技术规划

  • 总线舵机迁移：当前云台使用 MG996R PWM 舵机驱动，正推进迁移至幻尔（Hiwonder）串行总线舵机（HTS-25L + HX-08L）。总线舵机支持「手拉手」串联走线与实时状态回传，能够大幅简化内部布线并实现堵转保护。目前已完成控制板的基础通信调试，下一步需要适配官方的指令协议。
  • 端侧 AI 推理：计划将人脸检测模型从云端迁移至 ESP32-S3 端侧运行，利用其 Wi-Fi 与 AI 加速能力实现更低延迟的离线视觉追踪。
  • 表情系统升级：用更加细腻的 UI 动画和拟人化面部表情（眨眼、开心、思考等），让冰冷的硬件具备真实的情绪反馈。
