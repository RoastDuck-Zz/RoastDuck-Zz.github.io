# 🚀 项目经历

# 五自由度焊接机器人（毕业设计）—— 独立开发

基于 ESP32-S3 搭建「端云协同」控制系统，使用 SolidWorks 完成整机 3D 建模与物理样机 3D 打印。独立完成 5 自由度关节的正逆运动学解析求解，实现末端轨迹规划。引入 YOLOv8 深度学习模型重构视觉感知系统，通过实时采集图像并提取焊缝特征，结合 PID 算法实现了高精度闭环视觉伺服控制与自适应寻迹。

# 基于 ROS 的机器人系统课程设计 —— 团队负责人 & 核心开发

带领 7 人团队基于 ROSMASTER X3 平台搭建「树莓派 4B + STM32F103 扩展板」主从控制架构。独立制定 UART 串口数据帧协议，编写下位机电机驱动与 PID 控制程序。编写 ROS 通讯节点打通上下位机数据流，部署 Gmapping 建图与 TEB 局部路径规划算法，结合 OpenCV 实现视觉目标追踪与动态避障。

# 第三届山东省机器人及人工智能大赛（省三等奖）—— 机械与控制核心开发

负责全地形底盘的选型与机械结构设计，优化底盘重心与传动布局。基于 Arduino UNO 编写底层控制程序，主导多传感器数据采集与电机驱动模块的软硬件联调。

# 🛠 技术栈
